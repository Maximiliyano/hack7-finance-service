# hack7-finance-service


<h2>Main point</h2>

Розробити сервіс для слідкування за фінансами користувача.
Сервіс повинен дозволяти користувачу вести облік доходів, витрат, кредитів та депозитів. 
Він також повинен мати функціонал для відображення звіту про фінансовий стан користувача.

<h2>Requirements:</h2>

- Проєкт повинен бути розміщений на хостингу (веб-додаток) або мати посилання на відповідний файл
- Генерація звіту про фінансовий стан користувача у вигляді окремої сторінки, з можливістю фільтрації за різними параметрами (наприклад, за періодом часу, типом операції тощо) 
- Можuивістe додавання, видаuення та редагування доходів, витрат, кредитів та депозитів 
- Наявністe опису проєкту з README файлом та розміщення на репозиторії 
- Якщо це можuиво, використати хоча б одну API (в описі вказати, яке API використовували)

<h2>Criteria:</h2>

- Відповідність функціоналу вимогам вище
- Якість реаuізації та дизайн інтерфейсу
- Наявність та якісної документації по проєкту
- Використання та наuежна реаuізація API
- Розміщення проєкту на репозиторії, або посиuання на відповідний файл.
